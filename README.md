# Student Grade Tracker

A Java Swing based desktop application developed as part of the CodeAlpha Java Programming Internship.

## Features

- Add Student Name and Roll Number
- Enter Marks for 5 Subjects
- Calculate Total Marks
- Calculate Average
- Calculate Percentage
- Find Highest & Lowest Marks
- Display Grade
- Display Pass/Fail Result
- Save Report as TXT File
- Simple and User-Friendly Java Swing GUI

## Technologies Used

- Java
- Java Swing
- File Handling
- OOP Concepts
- VS Code
- Git & GitHub
